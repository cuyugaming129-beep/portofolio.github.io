# Template Web Portofolio

Template ini berisi halaman portofolio statis dengan bagian profil, pengalaman, projek, keahlian, dan kontak.

## Cara pakai

1. Buka `index.html` di browser.
2. Edit teks di `index.html` sesuai data Anda.
3. Ganti foto di folder `assets` dengan nama file berikut:
   - `foto-profil.jpg`
   - `foto-keahlian.jpg`
   - `projek-1.jpg`
   - `projek-2.jpg`
   - `projek-3.jpg`
4. Jika nama file foto berbeda, ubah bagian `src="assets/nama-file.jpg"` di `index.html`.

## File

- `index.html`: struktur halaman.
- `style.css`: tampilan dan responsif.
- `script.js`: menu mobile.
